# BrickSmart

Live Demo:
https://bricksmart.mishradigitalsolutions.com/

## Overview

BrickSmart is a web application built using Laravel, HTML, CSS, JavaScript, and AJAX. It demonstrates dynamic data handling without page reloads.

## Tech Stack

- Laravel
- PHP
- MySQL
- JavaScript
- AJAX
- HTML
- CSS

## Features

- Dynamic CRUD operations using AJAX
- Real-time data updates
- Responsive UI
- Laravel MVC architecture

## Purpose

This project demonstrates core Laravel and AJAX integration skills.
